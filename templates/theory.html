Python Dictionaries (Unit 2)

1. Conceptual Framework

Definition: A dictionary is an unordered, mutable, and indexed by keys collection. Unlike lists, which use positional indices (0, 1, 2...), dictionaries use Key-Value pairs.

Mapping: It is often called a "Mapping" or "Associative Array" because it maps a unique Key to a specific Value.

Syntax: Defined using curly braces {}. Items are separated by commas, and keys are separated from values by colons :.

my_dict = {"RollNo": 1, "Name": "Riya", "Marks": 95}

Properties of Dictionaries

Feature Rule
Keys Must be Unique and Immutable (e.g., Strings, Numbers, or Tuples).
Values Can be of any data type and can be Duplicated.
Ordering Unordered (Though modern Python ≥3.7 maintains insertion order, logically accessed by key).
Mutability The dictionary is mutable; you can add, remove, or change items after creation.

Essential Methods & Functions

🛠️ dict()

Constructor to create a new dictionary.

Code Example:
d = dict(name='Amit', age=16)
print(d)
Expected Output:
{'name': 'Amit', 'age': 16}
🛠️ len(d)

Returns the number of items in the dictionary.

Code Example:
d = {'a': 1, 'b': 2, 'c': 3}
print(len(d))
Expected Output:
3
🛠️ d.keys()

Returns a view object of all keys.

Code Example:
d = {'a': 1, 'b': 2}
print(list(d.keys()))
Expected Output:
['a', 'b']
🛠️ d.values()

Returns a view object of all values.

Code Example:
d = {'a': 1, 'b': 2}
print(list(d.values()))
Expected Output:
[1, 2]
🛠️ d.items()

Returns a view of (key, value) tuples.

Code Example:
d = {'a': 1, 'b': 2}
print(list(d.items()))
Expected Output:
[('a', 1), ('b', 2)]
🛠️ d.update(d2)

Merges dictionary d2 into d. Overwrites existing keys.

Code Example:
d = {'a': 1}
d2 = {'b': 2, 'a': 3}
d.update(d2)
print(d)
Expected Output:
{'a': 3, 'b': 2}
🛠️ d.get(key, default)

Safely fetches a value. Returns default if not found.

Code Example:
d = {'a': 1}
print(d.get('b', 'Not Found'))
Expected Output:
Not Found
🛠️ del d[key]

Removes a specific key-value pair.

Code Example:
d = {'a': 1, 'b': 2}
del d['a']
print(d)
Expected Output:
{'b': 2}
🛠️ d.clear()

Removes all elements, leaving an empty dictionary.

Code Example:
d = {'a': 1, 'b': 2}
d.clear()
print(d)
Expected Output:
{}
